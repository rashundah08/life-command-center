How Action Items Are Classified
GoalThe what — desired outcome. Keywords: reach, earn, save, lose, become, complete, achieve, grow, read, travel, visit. Contains Projects and Tasks.
ProjectThe how — multi-step effort. Keywords: plan, design, launch, build, set up, organize, redesign, implement, create, develop. Contains Tasks, links to Goals.
TaskThe do — single discrete action. Binary: done or not done. Rolls up to Projects and Goals.
Use Reclassify to cycle through types after analyzing.
Quick Classification Test
🎯 Goal — "WHAT do I want to achieve?"
☐ Outcome-based (a result or vision)?
☐ Measurable and time-bound?
☐ Months/years to complete?
☐ Needs multiple projects?
📁 Project — "HOW will I make it happen?"
☐ Clear deliverable or product?
☐ Weeks/months to complete?
☐ Rolls into a Goal?
☐ Made up of multiple tasks?
✅ Task — "DO it. Single action."
One action. Binary: done or not done. Can be completed in one session. No sub-steps.
